# HOLOdeLIVE ゲームルール

## 概要

2人専用ターン制カードゲーム。ラウンド制で、規定勝利数に達したプレイヤーが勝者となる。

- **プレイヤー数**: 2人
- **最大ラウンド数**: 5（デフォルト）
- **勝利条件**: 先に3ラウンド勝利（デフォルト）

---

## フィールド構成

### プレイヤー固有エリア

- **ステージ (Stage)**: カードを最大3枚、表向きに配置するエリア。3枚揃うとライブ準備状態になる。
- **楽屋 (Backstage)**: カードを1枚配置するエリア。ここに出すカードは裏向き（相手から不明）。

### 共有エリア

- **デッキ (Deck)**: 約60枚のユニークなカードで構成。場に置かれ、ドロー元となる。
- **自宅 (Home)**: 両プレイヤー共有のカード置き場。スキル等で送られたカードが一時的に留まる「墓地」的な役割。各ターン終了時に5枚を超えると、古いカードから順にゲームから除外される。

---

## カード (Card)

全てのカードにキャラクターが1人描かれている。デッキ内にカードの重複はない。

### プロパティ

| プロパティ | 説明 |
|---|---|
| **名前 (nickname)** | キャラクター名 |
| **アイコン (icons)** | キャラクターの性質を表す印。複数所持。12種: SEISO, CHARISMA, OTAKU, VOCAL, ENJOY, REACTION, DUELIST, KUSOGAKI, INTEL, SEXY, ALCOHOL, TRICKSTER |
| **スート (suits)** | キャラクターのカラー。複数設定。6種: LOVELY, COOL, HOT, ENGLISH, INDONESIA, STAFF |
| **スキル (skills)** | カード固有の効果。下記3種類に大別される |

### スキルの種類

| 種類 | 発動タイミング |
|---|---|
| **プレイスキル (Play Skill)** | カードをステージにプレイした時、またはオープンした時に発動 |
| **アクションスキル (Action Skill)** | アクションフェーズに任意で発動（各スキルにつき1回/ターン） |
| **常時スキル (Passive Skill)** | 常時効果を発揮 |

### スキルの特性

- **全カードがユニークなスキルを持つ**。共通化はほぼ不可能であり、各カード・各スキルに固有のコードが必要。
- **プレイヤーに選択を迫るスキルが存在する**。例: お互いのカードを交換するために、それぞれが手札の1枚を指定する。スキル解決中にプレイヤーの入力待ちが発生しうる。

---

## ゲームの流れ

### ゲーム開始

1. コイントスで先攻・後攻を決定
2. 各プレイヤーはデッキから2枚ドローして手札とする

### ターンの流れ

```
ターン開始
  │
  ├─ ライブ準備状態の場合 → ライブ発生（ドロー前）
  │
  ▼
デッキから1枚ドロー
  │
  ▼
アクションフェーズ
  │  - ステージ/楽屋の表向きカードのアクションスキルを任意で発動（各1回）
  │  - 楽屋の裏向きカードを表向きにできる（オープン）
  │    → オープン時、プレイスキルが発動
  │    → 続けてアクションスキルも発動可能
  │
  ▼
プレイフェーズ
  │  - 手札から1枚を空きステージまたは楽屋に出す（必須）
  │    - ステージ: 表向き → プレイスキル発動（プレイする）
  │    - 楽屋: 裏向き → プレイスキル発動しない
  │  - 手札がないか、スキルの影響で出せない場合はスキップ
  │
  ▼
ターン終了チェック
  │  - 自宅が5枚超 → 古い順に除外
  │  - ステージ3枚が全て埋まっている → ライブ準備状態に移行
  │
  ▼
相手のターンへ
```

### ライブ (Live)

ライブ準備状態のプレイヤーの次のターン開始時（ドロー前）にライブが発生する。

- **相手がライブ準備状態でない場合**: 自動的にラウンド勝利
- **相手もライブ準備状態の場合**: ショウダウンへ

### ショウダウン (Showdown)

各プレイヤーのユニットで勝負する。

- **ユニット (Unit)** の構成:
  - ステージの3枚（必須）
  - 楽屋のカード（表向きの場合のみ参加）
  - → 最大4枚
- **ランク決定**: ユニット内のアイコンとスートの組み合わせから、以下のランク表に基づいて決まる
- **同ランクの場合**: 先にライブ準備状態になったプレイヤーが勝利

### ランク表（高い順）

| ランク | 名前 | 条件 |
|---|---|---|
| 1 | **ミラクル (Miracle)** | 同一アイコン×同一スートのカードが3枚 |
| 2 | **トリオ (Trio)** | 同一アイコンのカードが3枚 |
| 3 | **フラッシュ (Flash)** | 同一スートのカードが3枚 |
| 4 | **デュオ (Duo)** | 同一アイコンのカードが2枚 |
| 5 | **カジュアル (Casual)** | 役なし |

### ラウンド終了処理

1. ステージのカードはゲームから除外される
2. 楽屋のカードは自分のステージへ移動する（裏向きのまま維持）
3. ラウンド勝者を記録
4. 次のラウンドは負けたプレイヤーの先攻で開始
